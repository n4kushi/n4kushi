## 📌 About Me
I am a **Systems Developer and Cybersecurity Researcher** with focused tracks in **Machine Learning** and **Trading Systems**. My work spans low-level software architecture, protocol-level security auditing, data-driven predictive modeling, and automated execution pipelines for financial markets.

* 🦀 **Systems & Concurrency:** Advanced implementation of memory-safe, asynchronous architectures using Rust and intensive low-level control with C++.
* 🤖 **Machine Learning:** Engineering data pipelines, feature extraction layers, and predictive model architectures for data-driven insights.
* 📈 **Trading Systems:** Designing automated market execution pipelines, programmatic middleman systems, and transaction engines.
* 🛡️ **Security Auditing:** Rigorous vulnerability mapping, smart contract risk mitigation, and network protocol verification under custom Linux deployment profiles.
